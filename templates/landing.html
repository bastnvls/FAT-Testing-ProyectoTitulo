¿Qué son las Pruebas FAT?

Las Pruebas de Aceptación en Fábrica (FAT - Factory Acceptance Test) son procedimientos de verificación que se realizan a equipos de red antes de su entrega al cliente. Estas pruebas garantizan que cada dispositivo cumple con las especificaciones técnicas y está listo para su implementación en producción.

Durante una prueba FAT, se ejecutan múltiples comandos CLI en el equipo, se verifica su funcionamiento, se documenta el estado del hardware y software, y se genera un informe profesional que certifica que el dispositivo pasó todas las verificaciones requeridas.

Cómo Funciona FAT Testing

Proceso simple en 3 pasos para automatizar tus pruebas FAT

1

Descarga el Software CLI

Accede a tu cuenta y descarga el ejecutable para Windows. Conéctalo por consola al equipo Cisco que deseas probar.

2

Ejecuta las Pruebas Automáticas

El software ejecuta automáticamente todos los comandos CLI necesarios según el modelo del dispositivo y captura las evidencias.

3

Genera el Informe

Accede a la plataforma web, sube los datos de la prueba y las imágenes. Obtén un informe Word profesional en segundos.
